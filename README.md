# Features

**Add Expenses:** Record expenses with date, amount, category, subcategory, and notes

**View All Expenses**: Display complete list of all tracked expenses

**Delete Entries**: Remove specific expense records by ID

**Category Organization**: Organize expenses into categories (Food, Travel, Sports, Shopping, etc.) with subcategories

**Monthly Summaries**: Calculate spending totals for specific time periods

**Shared Expenses**: Track who paid for group expenses with notes
